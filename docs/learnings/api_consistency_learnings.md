## Common FHIR Patterns

### Patient ID Validation
FHIR IDs must match: `^[A-Za-z0-9\-.]{1,64}$`

### Date Format
Always use ISO 8601: `2025-02-21` or `2025-02-21T10:30:00Z`

### Resource Types
Common types: Patient, Observation, Encounter, Appointment
